This is a minimal, elegant UI built with React and Tailwind that connects to the backend API. It includes:
- Home with hero and featured products
- Shop with search and price filters
- Product Detail
- About & Contact
- Auth (Login/Register)
- Admin Dashboard (products list, users list, create product)

Design Tokens
- Background: #F9F9F9
- Card: #EDEDED
- Primary/Dark: #2F343A
- Accent: #8E96AA
- Font: Inter, rounded corners (rounded-2xl)

Env
- Set VITE_BACKEND_URL to your API base (e.g., http://localhost:8000)

Run
- npm install
- npm run dev
